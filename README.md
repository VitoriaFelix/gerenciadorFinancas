## Clonar o repositório
```bash
$ git clone https://github.com/VitoriaFelix/gerenciadorFinancas.git
```
## Ir para o repositório
```bash
$ cd gerenciadorFinancas
```

## Instalar dependências
```bash
npm install
```

## Executar a aplicação
```bash
$ npm start
```
